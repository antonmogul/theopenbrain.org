# 8.4 The Decoy Effect

## How Adding Options Changes Preference

Imagine choosing between two subscription plans:

**Basic**: 50 articles/month — $5
**Premium**: Unlimited articles — $15

Some users will choose Basic (value-conscious); others will choose Premium (heavy readers). Reasonable choices based on individual needs.

Now add a third option:

**Basic**: 50 articles/month — $5
**Standard**: 75 articles/month — $14
**Premium**: Unlimited articles — $15

The Standard plan is clearly worse than Premium (nearly the same price, far less access). Who would choose it? Almost nobody.

But here's the strange part: adding this "dominated" option increases Premium subscriptions dramatically. The Standard plan is a **decoy**—an option designed not to be chosen but to change preferences among other options.

## The Mechanism Explained

The decoy effect (also called the "asymmetric dominance effect" or "attraction effect") works through relative comparison:

### Before the Decoy

Basic vs. Premium is a tradeoff:
- Basic: Lower price, limited access
- Premium: Higher price, unlimited access

Users must weigh their own price sensitivity against their access needs. No option is clearly "better"—it depends on individual circumstances.

### After the Decoy

Standard vs. Premium is NOT a tradeoff:
- Standard: $14 for 75 articles
- Premium: $15 for unlimited

Premium is clearly better—only $1 more for vastly more access. Premium now "dominates" Standard.

This creates **asymmetric dominance**: Premium dominates Standard but doesn't dominate Basic. The presence of a dominated option (Standard) makes the dominating option (Premium) more attractive.

Psychologically, users can now make a "confident" choice: "Premium is obviously better than Standard." This confidence transfers to the overall decision, even though Premium vs. Basic remains a genuine tradeoff.

## The Economist Subscription Study

The decoy effect was memorably demonstrated by Dan Ariely using The Economist's subscription options:

**Original options**:
- Web only: $59
- Print only: $125
- Web + Print: $125

Wait—Print only and Web + Print cost the same? Who would choose Print only when you can get Web included for free?

Nobody would (and nobody did in Ariely's experiments). But that's the point:

**Without Print-only decoy**: 68% chose Web only, 32% chose Web + Print
**With Print-only decoy**: 16% chose Web only, 84% chose Web + Print

The dominated Print-only option shifted preferences dramatically toward the dominating Web + Print option—even though it was never chosen itself.

## The Attraction Effect vs. Compromise Effect

Two related phenomena:

### Attraction Effect (Decoy Effect)

Adding a dominated option increases preference for the dominating option. The decoy is clearly worse than one alternative.

### Compromise Effect

Adding an extreme option increases preference for the middle option. The new option isn't dominated—it's just extreme.

Example: Add a "Super Premium" at $50/month. Some users will now view the $15 Premium as a "reasonable middle ground" between the cheap Basic and expensive Super Premium.

Both effects involve adding options to shift preference, but through different mechanisms.

## Common Implementations

### Three-Tier Pricing

The ubiquitous three-tier pricing page often employs decoys:

**Budget tier**: Functional but limited
**Middle tier**: Usually highlighted, often a decoy target
**Premium tier**: Comprehensive features, sometimes strategically priced near middle tier

The structure guides users away from the cheapest option through comparison effects.

### Product Comparison Tables

Feature comparison tables create decoy-like effects:

- Features are selected to make certain options look better
- "Included" checkmarks on the target plan, "Not included" on alternatives
- Pricing structured so one option dominates in value per feature

### Feature Bundling

Adding features to create dominance:

**Without decoy**: Plan A ($20) vs. Plan B ($40)
**With bundled decoy**: Plan A ($20) vs. Plan A+ ($38, Plan A + minimal extras) vs. Plan B ($40)

Plan B now dominates Plan A+, making Plan B relatively more attractive.

## Ethical Considerations

The decoy effect raises genuine ethical questions:

### When Decoys Help

Decoys can serve users by:
- Clarifying value comparisons in complex decisions
- Reducing decision paralysis through clearer relative evaluation
- Highlighting genuinely better options users might have overlooked

**Example**: A decoy that highlights that a slight price increase provides substantial value helps users make more informed decisions.

### When Decoys Manipulate

Decoys manipulate when:
- They obscure rather than clarify value
- They push users toward options that don't serve their needs
- They exploit comparison heuristics to extract more money
- They create false confidence in choices users wouldn't otherwise make

**Example**: A decoy designed to maximize revenue without regard for user needs manipulates comparison psychology.

### The Transparency Test

Ask: If users understood how the decoy worked, would they:
- Appreciate the clarification? (ethical)
- Feel manipulated? (unethical)
- Make different choices? (suggests manipulation)

## Criteria for Distinguishing Helpful vs. Manipulative

### Helpful Decoy Use

- Decoy highlights genuine value differences
- Target option genuinely serves user needs
- User benefits from clearer comparison
- No deception about options or features

### Manipulative Decoy Use

- Decoy exists solely to inflate preference for revenue-maximizing option
- Target option is overpriced for what it provides
- User would choose differently with complete information
- Artificial options created specifically for decoy effect

## Design Guidelines

### If Using Decoys

**Ensure genuine value**: The "winning" option should genuinely serve users, not just extract maximum revenue

**Maintain option quality**: Even the decoy should be a legitimate, deliverable option. Fake options cross ethical lines.

**Consider diverse users**: A decoy-pushed option that's good for some users may be wrong for others. Don't assume homogeneous needs.

### Alternatives to Decoys

**Direct comparison aids**: Help users compare without manipulated option sets
**Recommendation engines**: Suggest options based on stated needs rather than comparison effects
**Clear value articulation**: Explain what users get, letting them decide without psychological tricks
**Transparent pricing**: Price based on value delivered, not comparison psychology

## The Broader Context

The decoy effect is one of many comparison-based biases:

- **Anchoring**: First numbers influence subsequent judgments
- **Framing**: Presentation affects perceived value
- **Contrast effects**: Adjacent options affect perception
- **Reference dependence**: Evaluation is relative, not absolute

These biases aren't bugs—they're features of how cognition handles complex comparison. But they can be exploited.

## Summary

The decoy effect is real, powerful, and routinely used:

1. **Dominated options shift preferences**: Adding an option nobody wants can change what people choose.

2. **The mechanism is relative comparison**: We evaluate options against each other, not absolutely. Dominance relationships affect preference.

3. **Three-tier pricing exploits this**: Many pricing pages are structured around decoy effects.

4. **Ethical use clarifies value**: Decoys can help users understand genuine value differences.

5. **Manipulative use extracts money**: Decoys can push users toward options that don't serve their needs.

6. **Transparency matters**: Users who understand the decoy effect can evaluate whether it's helping or manipulating them.

The decoy effect reveals how deeply comparative our decision-making is. Designers who understand it face a choice: use comparison psychology to serve users, or exploit it at their expense.
